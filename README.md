# Projeto de Ponto de Venda (PDV)

## Descrição

Este projeto é um aplicativo móvel para um sistema de ponto de venda (PDV) desenvolvido em Java usando IntelliJ IDEA como IDE principal.

## Tecnologias e Ferramentas Utilizadas

- Java
- IntelliJ IDEA
